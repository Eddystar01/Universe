# Universe Software Requirements Specification (SRS)

Version: 1.0
Project: Universe
Codename: Project Orion

---

# 1. Vision

Universe is a multi-tenant SaaS platform that enables higher institutions to digitally manage attendance using secure QR codes, GPS verification, role-based authentication, analytics, and administrative tools.

The platform consists of:

- Laravel API
- Flutter Mobile Application
- React Admin Dashboard

Universe aims to reduce attendance fraud while simplifying academic administration.

---

# 2. Objectives

- Digital attendance
- Secure authentication
- Multi-university support
- Analytics and reporting
- Subscription management
- Notification system
- Offline-capable mobile application
- High scalability

---

# 3. Primary Users

- Students
- Lecturers
- Department Administrators
- University Administrators
- Super Administrators
